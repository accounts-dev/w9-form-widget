IRA Accounts (Final System Specification)

Scope

This logic applies exclusively to investors who select Account Type = IRA.


0. Entry Question (First Screen)

Question 0 (required)

What type of account did you use to invest?

Type

  • Dropdown (single select)

Options

  • Individual
  • IRA
  • Trust
  • LLC
  • Corporation
  • 401K

Rules

  • Required field (user cannot continue without selecting one)
  • The selection determines which W-9 flow/logic is shown next:
  • If IRA → load the IRA W-9 logic flow we defined
  • If Individual / Trust / LLC / Corporation / 401K → route to their specific logic (to be defined)

Data behavior

  • Store selected value as account_type
  • All downstream questions and auto-fill rules depend on account_type

1. Account Type (Pre-condition)

  • If account_type != IRA → this logic does not apply
  • If account_type == IRA → activate the flow below

2. Custodian Selection

Question 1 (required)

Who is the custodian of your IRA account?

Type

  • Dropdown (single select)

Options

  • Equity Trust
  • IRA Innovations
  • IRA Financial
  • Other (not listed)

Rules

  • Required field
  • If Other is selected:
  • Display an additional required field for custodian name

3. Investor Name

Question 2 (required)

What is your first and last name?

Type

  • Text input

4. W-9 Auto-Fill Rules (Name & Classification)

Line 1 — Name

Auto-fill with:

[Custodian Name]

Line 2 — Business name / disregarded entity

Auto-fill with:

FBO [First Name] [Last Name] IRA

Part III — Federal Tax Classification

  • Automatically select:
  • ☑ Other

Auto-fill the text field with:

IRA

All fields above must be:

  • Auto-filled
  • Non-editable by the investor

5. Custodian Address (W-9)

Logic

The W-9 address must be the custodian’s address.

Custodian from the predefined list

  • Address is stored in the system
  • Auto-fill:
  • Street
  • City
  • State
  • ZIP
  • Fields are not editable

Custodian = Other

  • Display manual address fields
  • All fields are required
  • Fields are editable

6. IRA Account Number

Question 3 (required)

What is your IRA account number?

Type

  • Text input

Usage

  • Auto-fill the Account number section of the W-9 form

Rules

  • Required field
  • No tax logic associated
  • For reference purposes only

7. IRA EIN

Question 4 (required)

What is your IRA EIN?

Rules

  • This must be the EIN specific to the IRA
  • Do not use the custodian’s general EIN
  • Basic EIN format validation is required

Support

  • Display a link or button to an instructional video if the investor does not yet have an EIN

8. Signature & Date

Final Step (required)

  • Electronic signature
  • Date (auto-filled with current date or selectable)

Rules

  • Both fields are required to complete the W-9
  • Without signature and date, the W-9 is considered incomplete

9. General Rules

  • Never use:
  • Investor SSN
  • Custodian’s general EIN
  • The following W-9 fields must be system-controlled (non-editable):
  • Line 1
  • Line 2
  • Federal tax classification
  • Address
  • Investor-provided inputs are limited to:
  • Name
  • IRA account number
  • IRA EIN
  • Custodian details (if “Other”)
  • Signature

10. Final Flow Order

  1. Account Type = IRA
  2. Select Custodian
  3. Enter First & Last Name
  4. Auto-fill W-9:
  • Line 1 (Custodian name only)
  • Line 2 (FBO Investor name)
  • Tax Classification (Other → IRA)
  1. Auto-fill Custodian Address
  2. Enter IRA Account Number
  3. Enter IRA EIN
  4. Signature + Date

This logic applies exclusively to investors who select Account Type = Individual on the first screen.

  • Text input

[First Name] [Last Name]

Rules

  • Required field
  • W-9 Line 1 should be system-controlled (non-editable)

3. Personal Address

Question 2 (required)

What is your personal address?
 (Instruction must explicitly state this is the investor’s personal/home address.)

  • Address fields (required):

  • City
  • State
  • ZIP

  • Auto-fill the W-9 Address section with the provided address

4. SSN

Question 3 (required)

What is your Social Security Number (SSN)?

Type

  • Text input

W-9 Mapping

  • Auto-fill the W-9 field:
  • Social Security number

Rules

  • Required field
  • Basic SSN format validation
  • W-9 SSN field should be system-controlled (non-editable)

5. W-9 Federal Tax Classification (Part III)

Auto-fill rule (triggered by account type)

If account_type = Individual, then in Part III:

  • Automatically check:
  • ☑ Individual/sole proprietor

Rules

  • Checkbox is system-controlled (non-editable)

6. Signature & Date (Final Step)

Required

  • Electronic signature
  • Date (auto-filled with current date or selectable)

Rules

  • Both required to complete the W-9
  • Without signature and date, the W-9 is considered incomplete

W-9 Logic — LLC Accounts (System Specification)

Scope

This logic applies exclusively to investors who select Account Type = LLC on the first screen.

1. Account Type (Pre-condition)

  • If account_type != LLC → this logic does not apply
  • If account_type == LLC → activate the flow below

2. LLC Tax Treatment (Routing Question)

Question 1 (required)

What type of LLC do you have?

Type

  • Dropdown (single select) or radio buttons

Options (exact)

  1. Disregarded LLC
     Helper text must be shown: “Typically single-member LLCs are disregarded unless they elect a different tax treatment.”
  2. C Corp
  3. S Corp
  4. Partnership

Rules

  • Required selection
  • This selection routes the user into one of two paths:
  • Path A: llc_type = Disregarded LLC
  • Path B: llc_type ∈ {C Corp, S Corp, Partnership}

Path A — Disregarded LLC

A1. Individual Name

Question A2 (required)

What is your first and last name?

W-9 Mapping

Auto-fill W-9 Line 1 (Name) with:

[First Name] [Last Name]

  • Line 1 is system-controlled (non-editable)

A2. LLC Name

Question A3 (required)

What is the legal name of your LLC?

W-9 Mapping

Auto-fill W-9 Line 2 (Business name / disregarded entity) with:

[LLC Legal Name]

  • Line 2 is system-controlled (non-editable)

A3. Part III — Federal Tax Classification (auto-fill)

For llc_type = Disregarded LLC, Part III must:

  • Automatically check:
  • ☑ Individual/sole proprietor or single-member LLC
  • Checkbox must be system-controlled (non-editable)

A4. Address

Question A4 (required)

What is your address?

Type

  • Address fields (required):
  • Street
  • City
  • State
  • ZIP

W-9 Mapping

  • Auto-fill W-9 Address section
  • Address fields on W-9 are system-controlled (non-editable) after input

A5. SSN (Required)

Question A5 (required)

What is your Social Security Number (SSN)?

W-9 Mapping

  • Auto-fill the Social Security number field on the W-9
  • Field is system-controlled (non-editable)

Validation

  • Basic SSN format validation

A6. LLC EIN (Required)

Question A6 (required)

What is your LLC EIN?

W-9 Mapping

  • Store/collect EIN as required data point for the workflow
  • (If your W-9 rendering supports storing both, keep it associated with the investor record; do not override SSN field.)

Validation

  • Basic EIN format validation

A7. Signature & Date (Final Step)

  • Electronic signature (required)
  • Date (auto-filled with current date or selectable) (required)
  • Without signature/date → W-9 is incomplete

Path A — Final Flow Order

  1. Account Type = LLC
  2. Select LLC type = Disregarded LLC
  3. Enter investor first & last name → auto-fill W-9 Line 1
  4. Enter LLC legal name → auto-fill W-9 Line 2
  5. Auto-check Part III: Individual/sole proprietor or single-member LLC
  6. Enter address → auto-fill W-9 Address
  7. Enter SSN → auto-fill W-9 SSN field
  8. Enter LLC EIN (collect/store)
  9. Signature + Date

Path B — LLC taxed as C Corp / S Corp / Partnership

B1. Part III — Federal Tax Classification (auto-fill)

For llc_type ∈ {C Corp, S Corp, Partnership}, Part III must auto-select the matching classification:

  • If llc_type = C Corp → check ☑ C Corporation
  • If llc_type = S Corp → check ☑ S Corporation
  • If llc_type = Partnership → check ☑ Partnership

All are system-controlled (non-editable).

B2. LLC Name

Question B2 (required)

What is the legal name of your LLC?

W-9 Mapping

Auto-fill W-9 Line 1 (Name) with:

[LLC Legal Name]

  • Line 1 is system-controlled (non-editable)

B3. Address

Question B3 (required)

What is the LLC’s address?

Type

  • Address fields (required):
  • Street
  • City
  • State
  • ZIP

W-9 Mapping

  • Auto-fill W-9 Address section
  • Fields on W-9 are system-controlled (non-editable) after input

B4. LLC EIN

Question B4 (required)

What is your LLC EIN?

W-9 Mapping

  • Auto-fill the Employer Identification Number (EIN) field on the W-9
  • Field is system-controlled (non-editable)

Validation

  • Basic EIN format validation

B5. Signature & Date (Final Step)

  • Electronic signature (required)
  • Date (auto-filled with current date or selectable) (required)
  • Without signature/date → W-9 is incomplete

Path B — Final Flow Order

  1. Account Type = LLC
  2. Select LLC type (C Corp / S Corp / Partnership)
  3. Auto-check Part III based on LLC type
  4. Enter LLC legal name → auto-fill W-9 Line 1
  5. Enter LLC address → auto-fill W-9 Address
  6. Enter LLC EIN → auto-fill W-9 EIN field
  7. Signature + Date
